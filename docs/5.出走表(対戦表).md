# 5. 出走表(対戦表)

## 5.1. URL
~~~
https://keirin.netkeiba.com/race/match_list/?race_id={raceId}
~~~

{raceId} = {YYYYMMDD}{JYO_CD}{レース番号:2桁}

例: 202603034412
    - 2026年03月03日
    - 大垣競輪場（コード: 44）
    - 第12レース

## 5.3. データ
**出走表(対戦表)の要素**

| 項目名                             | データ設定例          |
| ---------------------------------- |-----------------------|
| レースID                           | RRRRRRRR              |
| 車立配列 ※最大配列数:9(4～9)      |                       |
|   枠番                             |   5                   |
|   車番                             |   6                   |
|   選手名                           |   小松崎大地          |
| ---------------------------------- |-----------------------|
|   総合                             |   2-10                |
|   1番車対戦                        |   0-3                 |
|   2番車対戦                        |                       |
|   3番車対戦                        |   0-1                 |
|   4番車対戦                        |   1-4                 |
|   5番車対戦                        |   0-1                 |
|   6番車対戦                        |   －                  |
|   7番車対戦                        |                       |
|   8番車対戦                        |   0-1                 |
|   9番車対戦                        |   1-0                 |
| 登録日時                           | yyyy-mm-dd hh:mm:ss   |

## 5.4. 特記事項
- 1車番対戦～9番車対戦のデータが未設定の場合がある（対戦履歴なし）
- 自車番と同番の対戦データは「-」が設定されている
