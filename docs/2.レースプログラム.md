## 2.3. データ
**プログラムの要素**

| 項目名                             | データ設定例            |
| ---------------------------------- |-------------------------|
| 開催ID                             | XXXXXXXX                |
| 開催配列 ※最大配列数:15(通常は3～6)|                        |
|   開催日                           |   2026-02-28            |
|   グレード（GP,G1,G2,G3,F1,F2）    |   GP                    |
|   開催区分（※1）                  |                         |
|   プログラム種別                   |   初日                  |
|   プログラムID ※ユニークキー      |   KKKKKKKK              |
| 登録日時                           | yyyy-mm-dd hh:mm:ss     |


**レースプログラムの要素**

| 項目名                             | データ設定例            |
| ---------------------------------- |-------------------------|
| プログラムID                       | KKKKKKKK                |
| レース配列 ※最大配列数:24(通常は7～12) |                    |
|   レース番号                       |   12                    |
|   レースタイトル                   |   Ｓ級 初日特別選抜     |
|   車立数                           |   9                     |
|   発走時刻                         |   16:20                 |
|   締切時刻                         |   16:15                 |
|   レースID ※ユニークキー          |   RRRRRRRR              |
| 登録日時                           | yyyy-mm-dd hh:mm:ss     |

※1:モーニング, ナイター, ミッドナイト, ガールズ

1.4. 特記事項
- 開催区分は「https://keirin.netkeiba.com/race/course/calendar.html?kaisai_year={YYYY}&kaisai_month={MM}&jyo_cd={JYO_CD}&kaisai_type=1
」から取得する
- 開催区分が設定されていない場合がある（通常開催の場合）
- 開催区分が2つ設定されている場合がある（モーニング + ガールズ、ナイター + ガールズ、ミッドナイト + ガールズ）
