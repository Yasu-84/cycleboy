# 4. 出走表(直近成績)

## 4.1. URL

https://keirin.netkeiba.com/race/entry/results.html?race_id={raceId}

{raceId} = {YYYYMMDD}{JYO_CD}{レース番号:2桁}

例: 202603034412
    - 2026年03月03日
    - 大垣競輪場（コード: 44）
    - 第12レース

## 4.4. 特記事項
- サイトには、直近30までデータが存在するが、取得するデータは直近3までとする
- 開催初日や補充要員など、今節のデータが存在しない場合がある
- 今節～直近3のレース配列数は異なる場合がある
